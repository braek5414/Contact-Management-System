# Contact Management System

A simple Python-based **Contact Management System** that runs in the terminal.  
This project demonstrates file handling, user interaction, and basic CRUD operations using a plain text file.

---

## Overview

This program allows users to store and manage contact information locally.  
Each contact includes a **name**, **email**, and **phone number**, all stored in `contact.txt`.

The system provides a simple menu that lets users:

- Display all saved contacts  
- Search for a specific contact  
- Add a new contact  
- Delete an existing contact  
- Modify contact information  
- Quit the program  

---

## Features

### **Show Contacts**
Displays all contacts currently saved in the `contact.txt` file.  
Each contact is shown as:

Name: <name>
Email: <email>
Phone: <phone>
